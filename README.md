### Running Go Program

#### Just Run code

```bash
go run <file name>.go
```

#### Compile code into executable in dir

```bash
go build
```

 will create binary file with the name of folder

 run executable

```bash
./go-examples
```

#### Executable put into bin folder

```go install```
